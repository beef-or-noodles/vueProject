## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

#文件夹
tool 工具类
totalApi   接口文件
upload  图片上传文件夹
uploadApi   图片上传接口
db.js   数据库连接信息
main.js node服务器入口
sqlfun.js  sql语句
```
